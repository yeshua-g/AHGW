# AHGW

![MATLAB](https://img.shields.io/badge/MATLAB-R2023b%2B-blue) 

MATLAB folders with functions and a Live Script for the AHGW course assignment.

## Overview
This repository contains the material for the **AHGW** course assignment:

- MATLAB files (functions, scripts)
- The main Live Script: `AHGW_YeshuaGiacalone.mlx`  
  You can run it in MATLAB after downloading the folder.
- An exported PDF version: `AHGW_YeshuaGiacalone.pdf`  
  To quickly inspect the results **without running MATLAB**, directly from GitHub.  
  This PDF contains **everything** from the Live Script:
    - All the MATLAB **code**
    - All **plots and figures**
    - All **comments and explanations**
  
  It has been formatted in **A2** size so that everything fits in a compact number of pages, making it easy to read and review without scrolling through multiple screens in MATLAB.

## Download and run locally
You can download the entire folder to your computer in two ways:

### Option 1 — Download ZIP (via GitHub web)
1. Open the repository page on GitHub.  
2. Click **Code → Download ZIP**.  
3. Extract the ZIP on your computer.

### Option 2 — Clone with Git
```bash
git clone https://github.com/yeshua-g/AHGW.git
cd AHGW
```
